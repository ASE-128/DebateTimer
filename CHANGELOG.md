# Changelog

## [3.0.1] - 2026-07-10

### Fixed
- 修复编辑页背景预览不显示的问题（默认缩放值与层叠上下文修正）
- 修复"保存配置"按钮会自动打开计时页的问题，保存与进入计时页功能现在完全独立
- 修复 30 秒/5 秒/时间到提示音不一致的问题，统一为时间到铃声，次数分别为 1/2/3 次
- 补全 CHANGELOG：从 `previous version` 历史压缩包中提取 2.0.0 → 2.11.3 的变更记录

## [3.0.0] - 2026-07-10

### BREAKING CHANGES
- 配置 schema 升级到 `version: 3`；旧配置需通过迁移向导/自动迁移转换，原配置已备份。
- `backgroundImageSettings.scale` 已移除，请使用 `scaleX` / `scaleY`。
- 正式移除 WinRAR SFX 导出支持，独立计时器仅支持 NSIS 安装包。

### Added
- 集成 electron-updater 自动更新
- 编辑页更新提示与更新日志展示
- 跨平台构建配置（macOS dmg/zip、Linux AppImage）
- GitHub Actions 自动发布

### Changed
- 版本号与 NSIS 安装包元数据统一使用 package.json 中的真实版本号
- 优化安装包体积：清理 vendor/nsis 非必需目录，进一步裁剪 Electron 运行时

### Removed
- 不再支持 WinRAR SFX 导出（已在 2.11.x 迁移到 NSIS）
- 3.0.0 不保证 100% 向下兼容 2.x 配置

## [2.11.3]

### Changed
- 独立计时器导出机制由 WinRAR 自解压 EXE 改为 NSIS 安装程序 EXE
- 安装程序支持检测旧版本并提示卸载、用户自选安装目录
- 安装完成后自动创建桌面/开始菜单快捷方式并启动应用
- 项目内置 `vendor/nsis/` NSIS 编译器，用于应用内生成安装包

## [2.11.2]

### Changed
- NSIS 配置新增 `license` 与 `shortcutName: "辩论赛计时器"`
- 安装程序细节优化与修复

## [2.11.0]

### Added
- 所见即所得编辑界面：直接在计时器预览中点击/拖动编辑文本元素
- 对称文本元素（正反方）拖动时自动同步移动
- 状态栏支持高度拖拽、颜色/透明度/渐变自定义
- 背景支持纯色/渐变/图片，图片可调整透明度、缩放、XY 偏移，并以 Base64 编码存入配置

### Changed
- 导入配置兼容旧版配置；导出配置包含完整布局、状态栏与背景设置

## [2.0.0]

### Added
- 初始版本，基于 Electron 42.x 的桌面辩论赛计时器
- 支持赛事信息、环节管理（无计时/单边计时/双边对辩）、名称模板、主题样式、本地上传字体
- 支持导出配置为 JSON，以及导出独立 EXE
- 计时控制：启动/暂停、停止、重置、环节跳转、设置时间、切换持方、全屏、音效试播等
